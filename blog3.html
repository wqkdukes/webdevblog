9/13 - 9/17, Flex Box

This week we've been discussing Flexbox, which is a layout that allows you to effectively organize items in a container. Even when these items have unknown sizes or are dynamic. Some properties included in the flex layout are flex-direction, justify-content, and align-items. These properties would typically be used on a flex container to organize its contents.
